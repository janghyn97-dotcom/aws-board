# 자유게시판

JS 만든 동적 게시판 프로젝트입니다.

## 기능
- 게시글 목록 조회 (fetch로 posts.json 불러오기)
- 게시글 등록 (제목, 작성자 입력)
- 게시글 삭제
- 게시글 검색 (제목 기준 실시간 필터)
- 게시글 클릭 시 콘솔 로그 출력

## 실행 방법
1. VS Code에서 프로젝트 폴더 열기
2. Live Server 확장 설치
3. `list.html` 우클릭 → Open with Live Server
4. 브라우저에서 `127.0.0.1:5500` 접속 확인

## 사용 기술
- HTML5 / CSS3 (Flexbox)
- JavaScript (fetch, DOM 조작)
